# GHL Ask AI — Gig Announcement Post Prompt

Paste everything below the line into Ask AI. Reuse for every show on the run — swap the
SHOW line and the facts block for each date.

Create a Facebook post for a live show — two parts: (1) the post graphic, (2) the caption.

THE SHOW: Dixon Hall — live at Admiral's Cup, Baltimore, MD — Wednesday, July 8 — 7:00 PM.

PART 1 — THE GRAPHIC
Design it like a limited-run CONCERT POSTER someone would frame, not a social media template.
Square 1080×1080.

- Background: near-black warm brown (#0a0705) with heavy brick/grunge texture, deep vignette,
  one warm gold stage-light glow bleeding in from the top corner, subtle film grain.
- Faint ghosted American-flag stars worked into the texture, barely visible.
- Type hierarchy (top to bottom):
  1. Small-caps gold eyebrow, wide letter-spacing: "NO LIMIT COUNTRY · LIVE"
  2. Massive ultra-condensed uppercase "DIXON HALL" in metallic chrome — a silver-white
     gradient that reads like a polished belt buckle. This owns the poster.
  3. Gold uppercase line: "LIVE AT ADMIRAL'S CUP"
  4. Thin gold rule, then elegant small caps: "WED JULY 8 · 7:00 PM · BALTIMORE, MD"
- Add a ticket-stub style date block — "JUL 08" in a thin gold outlined box — anchored in a corner.
- Palette is ONLY: near-black #0a0705, warm cream #e7ddcb, antique gold #cda45f, chrome silver.
- Sharp corners everywhere. Distressed/letterpress texture on the big type.
- DO NOT use: rounded frames, glossy render style, emoji graphics, blue or purple, stock crowd
  photos, confetti, "You're invited" energy. If it looks like a Canva flyer, start over.
  It should look like band merch.

PART 2 — THE CAPTION
Voice: gritty, heartfelt, plain-spoken. Short lines. No corporate hype, no emojis,
no exclamation points. Sounds like a guy who means it, not a marketing team.

Structure:
1. One-line hook that sounds like a lyric or a dare.
   (Register examples so you match the tone: "Baltimore, bring your boots." /
   "Real country hits the harbor Wednesday night." — write 3 original options like these.)
2. Blank line. Then the facts, each on its own line:
   Admiral's Cup — Fells Point
   Baltimore, MD
   Wednesday, July 8 · 7:00 PM
3. One closer line with some grit — the "pull up a stool, we'll handle the rest" vibe.
4. Sign-off: — Dixon Hall · No Limit Country
5. Max 5 hashtags: #DixonHall #NoLimitCountry #LiveMusic #BaltimoreMusic #FellsPoint

Deliver 2 caption versions: one 40–60 words, one punchy version under 25 words.
